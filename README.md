## 🚀 Como usar

Cada pasta contém código funcional. Para executar:

```bash
# Clone o repositório
git clone https://github.com/adriano-viana/claude-code-demos.git

# Entre na demo desejada
cd claude-code-demos/video-01-criador-usa/secao-X-nome/

# Se tiver package.json, instale dependências
npm install

# Execute testes (se disponíveis)
npm test
```
